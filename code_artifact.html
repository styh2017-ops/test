핵심 개념 가이드 (교사용)

밀폐된 실린더 내부에서 일어나는 액체 $\rightleftharpoons$ 기체 상평형 모델입니다. 온도가 일정할 때 증발 속도는 항상 일정하지만, 응축 속도는 기체의 단위 부피당 입자 수(밀도/압력)에 비례합니다.

1. 부피 감소 (압력 증가): 순간적으로 밀도가 증가해 응축 속도가 급증하며, 기체 입자수가 감소해 새로운 평형에 도달합니다.
2. 부피 증가 (압력 감소): 순간적으로 밀도가 감소해 응축 속도가 급감하며, 증발이 우세해져 기체 입자가 다시 늘어납니다.
